== Installation ==

1. Upload the 'wp-image-altifier' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. That's it! The plugin will now automatically set the ALT attributes for images in your posts, pages, and custom posts.

== Frequently Asked Questions ==

= Does this plugin work with custom post types? =

Yes, WP Image Altifier works with custom post types as long as they use the 'the_content' filter.
